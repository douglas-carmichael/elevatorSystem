# Screenshots

Place the screenshots referenced by the top-level [`README.md`](../README.md)
here. Each of the app's four windows is captured twice — once with the UI in
English, once in French. Toggle the language with the `[EN]`/`[FR]` control in
the Group Dispatcher footer (or press `L`), then re-capture the same windows.

| File | Window | Language |
|------|--------|----------|
| `dispatcher.png`    | Group Dispatcher (VT320 2D control panel) | English |
| `dcl.png`           | DCL Terminal                              | English |
| `synoptic.png`      | Hoistway Synoptic (SceneKit 3D shaft)     | English |
| `dynamics.png`      | Cab Dynamics (velocity-trace scope)       | English |
| `dispatcher_fr.png` | Group Dispatcher (VT320 2D control panel) | French  |
| `dcl_fr.png`        | DCL Terminal                              | French  |
| `synoptic_fr.png`   | Hoistway Synoptic (SceneKit 3D shaft)     | French  |
| `dynamics_fr.png`   | Cab Dynamics (velocity-trace scope)       | French  |

Filenames are case-sensitive and must match the `README.md` image links
exactly (lowercase, `.png`, `_fr` suffix for the French set).
